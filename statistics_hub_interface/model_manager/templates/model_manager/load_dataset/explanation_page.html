How to Use This Page

Follow these steps to manage your datasets:

  • File Explorer: Use this tab to browse existing datasets.
    • Upload datasets from your computer.
    • Delete previously uploaded datasets.
    • Create folders to organize your datasets into various hierarchical levels.
    • Select the dataset you want to load by clicking on it.
  • Import Options: Configure import options for CSV files and preview the first row to adjust parameters. Similar options are available for XLSX, TAB, and JSON files.
  • MySQL: Use this tab to connect to a MySQL database and load datasets from it.
  • InfluxDB: Use this tab to connect to an InfluxDB database and load datasets from it.
  • Click "Prepare Dataset" to load the selected dataset, perform exploratory data analysis, and preprocess it with the specified options.
  • Click "Load Train" to load the training dataset and initiate the training process with the specified options.

Make sure to configure the options correctly before loading the dataset to ensure proper data preprocessing.
