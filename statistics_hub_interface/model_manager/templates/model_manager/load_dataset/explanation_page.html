How to Use This Page

Follow these steps to manage your datasets:

  • Navigate through the "File Explorer" tab to browse existing datasets. Click on folders to expand and view their contents.
  • Use the "Upload File" tab to add new datasets. You can upload files or create new folders to organize your datasets.
  • Select a dataset to preview its content in the preview section.
  • Configure the import options for CSV files using the options provided in the "CSV Options" tab.
  • Click "Load Dataset" to load and preprocess the selected dataset with the specified options.

Make sure to configure the options correctly before loading the dataset to ensure proper data preprocessing.
